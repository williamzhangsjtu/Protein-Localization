2020年春季机器学习课程大作业数据集介绍：
    本次大作业是使用机器学习的方法对人类蛋白质图谱进行多标签分类的任务

    1，train目录是训练集数据，train目录下共636个文件夹，每个文件夹下面有若干张 512x512 的人类蛋白质图谱的图片数据。
    2，每个文件夹里面的若干张图片具有相同的标签。
    3，train.csv是标签文件，每一行对应train目录里面的一个文件夹，每一行中，第一项是目录的名称，后面若干项是这个文件夹
    里面的图片数据对应的标签，由于是多标签分类问题，后面若干项可能会包含多个标签，相互之间用";"分隔。
    4，请同学们自行在训练集中划分验证集进行模型的调优。
    5，建议使用召回率，准确率，F1值以及AUC作为评价指标。
    6，测试集会在大作业提交截止前一天发布，届时同学们将使用自己的模型对测试集进行预测，并提交预测结果，助教将根据测试集
    的预测结果进行打分。
